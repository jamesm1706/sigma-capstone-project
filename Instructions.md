Once you run the code the main menu will appear, presenting the two modes you can choose between. Entering '1' into the terminal will run the single player game, while '2' will run the two player game. 

After a mode is selected the player will be prompted to enter their name, and in the two player mode the second player will then be asked for their name.

Once entered, the game will run and the game board will appear in the terminal. The numbers in each board square correspond to the entry required to play a move there. For example, entering '5' will play your move into the middle square.

The game ends when one player wins, by having three pieces in a row, either vertically, horizontally or diagonally. It will also end in a tie if the board is full with no winning lines.
